# Data Services

This is a Astro project that provide the start for publishing to end users data focused information.

It has custom components for 

- Data Graphs based on Cytoscape.js
- Entity Relationship Diagrams
- A front page to provide a search, into and starting point for navigation
- Document style pages with a number of doucment folders access throught a top menu item
- Guidlines to describe the way in which the system works
- Training to provide user will training on the data platform 


# Data Host

This is a Go & React project that provides the backend for the data services.

The starting point is a Directory of functionality within the system in terms of components
There is a menu bar at the top of the page for the theme and user profile & settings 


## Directory (Home) page 

This is a directory of functionality within the system in terms of components
The side bar contains a list of criteria to filter the directory
The main page has a series of cards to provide a starting point for navigation to the :-

- Internal Operations 
- Sidecar Operations run in different execututable as part of the product suite
- External system and sites 


## Platforms

Provide access to differnent platform in terms of 

- Environment details 
- Connection Management 
- issues with the connections

## Inspect 

Provides the ability review the schema files on the site

- View - Gives visability of the site
- Guidelines - shows the selection criteria for the guidlines in action
- - Allows the categories to be edited
- Training - shows the selection criteria for the training in action 
- - Allows the categories to be edited
- Schema Health - gives a check as statistics on the schema json files
- - Placeholder to view diagrams
